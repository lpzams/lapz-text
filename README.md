## 使用方法（智能聊天机器人）

1. 使用 Qt Creator 打开 `build` 文件夹中的 `.pro` 文件。
2. 修改 main.cpp 中的目录中自己的路径
3. 注意使用的版本为qt6.6.3,5几的版本需要自己更改相应内容
4. 请将音量调低，因为有较大音量的大貔貅（已经关闭）。
